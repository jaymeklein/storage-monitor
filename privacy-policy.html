Privacy Policy for Local Storage Monitor Chrome Extension

Data Collection

This extension does NOT collect, store, or transmit any user data to external servers.

Local Storage Access

The extension only accesses the browser's local storage for the specific purpose of:

  • Reading values from local storage based on user-specified keys
  • Displaying these values to the user in the extension popup
  • Allowing users to copy values to their clipboard

Data Storage

All data remains on your local device:

  • The monitored key is stored locally in Chrome's sync storage
  • Local storage values are only accessed when you actively use the extension
  • No data is sent to any external servers or third parties

Permissions

The extension requires:

  • storage: To save your preferred monitoring key
  • activeTab: To access local storage of the current website
